# Gradient Descent

f(x) = -1 \* (x² - 2x + 1)
導數 f'(x) 告訴我們這條線在某個點上是往上斜還是往下斜。
如果 f'(x) > 0，表示那個地方「往右邊上升」。
如果 f'(x) < 0，表示那個地方「往右邊下降」。
如果 f'(x) = 0，表示那裡「剛好平」，也可能是最高點或最低點。

# vecGradient.py

step 要小，是因為導數定義是「極小變化量的變化率」；
但不能太小，因為電腦有浮點精度限制。
0.01 是一個在準確與穩定之間的折衷值。

## f(p)

p 代表一個點

## df(f, p, k)

p 複製一份給 p1
k = 0 → 對 x 求偏導 (df/dx)
k = 1 → 對 y 求偏導 (df/dy)

## grad(f, p)

p 複製一份給 gp
迴圈對每個 k 求偏導，結果存進 gp
